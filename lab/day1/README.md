# Day 1 Programs

## [Question 1](./StudentDemo.java)

Write a program to print your name, roll no, section and branch in separate lines.

**Input:** `Mention Name,Roll no,Section and branch`

**Output:**
```
Display Name
Display Roll No
Display Section
Display Branch
```

## [Question 2](./GradeCalculator.java)

Write a program to print the corresponding grade for the given mark using if..else statements.

**Input:** `Mention the grade in the program`

**Output:** `Display the Grade either O/E/A/B/C/F`

## [Question 3](./Weekday.java)

Write a program to print the week day for the given day no. of the current month using switch case statement.

**Input:** `Mention the Day no in the program`

**Output:** `Display the week day (either Sunday/Monday/Tuesday/Wednesday/Thursday/Friday/Saturday)`
 
## [Question 4](./Palindrome.java)

Program to check a user entered number is palindrome or not.

**Input:** `Mention the number in the program`

**Output:** `Display the number is Palindrome or not`
 
## [Question 5](./Naming.java)

Write a program in Java to take first name and last name from user and print both in one line, as last name followed by first name.

**Input:**
```
Enter first name: KIIT
Enter Second Name: UNIVERSITY
```

**Output:** `UNIVERSITY KIIT`
